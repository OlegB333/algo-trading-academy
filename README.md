# Algo Trading Academy
> Инфраструктура алгоритмического бота на базе Freqtrade

Добро пожаловать в репозиторий курса по алгоритмической торговле. Вся основная обучающая информация находится на платформе курса. Этот репозиторий содержит готовую и настроенную инфраструктуру для работы.

## 🚀 Как начать работу

Вам не нужно вручную редактировать конфигурации. В редакторе встроен персональный ИИ-ассистент, уже обученный управлять этой инфраструктурой.

Просто откройте чат (Cmd/Ctrl + L) и напишите:
> **«Агент, запусти проект»**

Ассистент сам проверит Docker, настроит порты и поднимет интерфейс для работы.

## 📁 Доступные модули

Платформа поддерживает два независимых режима работы:

### 1. Базовый — Криптовалюты (Binance)
Работает из коробки. Команда агенту: *«Запусти проект»*.

### 2. Продвинутый — Фондовый рынок (Interactive Brokers)
Торговля акциями, форексом и фьючерсами через Interactive Brokers.

**Требования:**
- Установленный **Trader Workstation (TWS)** — официальный терминал IB
- Учётная запись Interactive Brokers (Paper Trading подходит для обучения)
- Включённый API в настройках TWS (порт 7497 для Paper Trading)

Команда агенту: *«Подключи Interactive Brokers»* — ассистент проверит TWS,
даст инструкцию по настройке API и убедится что соединение работает.
