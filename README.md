# Algo Trading Academy

Учебная платформа для создания алгоритмических торговых стратегий с ИИ-агентом.

Вам не нужно уметь программировать. Вы — **архитектор торговой системы**, ИИ-агент — ваш кодер.

---

## Что вы научитесь делать

### 📊 Строить стратегии
Описываете идею словами — агент переводит её в код.
Поддерживаемые подходы: трендовые, контртрендовые, пробои уровней, индикаторные системы, портфельные стратегии.

### 🧪 Тестировать на истории
```
Запусти бэктест стратегии CrossSMA на BTC за последние 3 месяца
```
Агент запускает бэктест и объясняет результаты: просадка, винрейт, профит-фактор.

### 📥 Работать с данными
```
Скачай исторические данные для ETH/USDT за 90 дней
```
Агент загружает котировки для нужных пар и таймфреймов.

### ⚙️ Настраивать параметры
Настройка торговых пар, таймфреймов, стоп-лоссов, объёма позиций — через диалог с агентом.

### 🛡️ Управлять рисками
Агент помогает настроить лимиты: максимальная просадка, количество открытых позиций, условия остановки бота.

### 🚀 Деплоить на сервер
```
Помоги развернуть бота на VPS
```
Агент помогает арендовать сервер, настроить Docker, перенести стратегию и запустить бота 24/7.

### 🔐 Настраивать безопасность
API-ключи биржи (без права вывода), защита VPS, изоляция среды через Docker.

### 📱 Получать уведомления в Telegram
Оповещения о сделках, ошибках, статусе бота — прямо в мессенджер.

---

## Как начать

Откройте проект в Antigravity и напишите агенту:

> Запусти проект

Агент сам настроит окружение и откроет веб-интерфейс FreqUI.

---

## Что уже настроено из коробки

| Компонент | Описание |
|-----------|----------|
| **Freqtrade** | Фреймворк для алгоритмической торговли |
| **dry-run** | Симуляция торговли (реальные деньги не используются) |
| **FreqUI** | Веб-интерфейс: сделки, графики PnL, статистика |
| **SampleStrategy** | Учебная стратегия на пересечении SMA |
| **Binance** | Подключена биржа (spot, без реальных ключей) |
| **ИИ-агент** | Настроен с правилами и скиллами для учебных задач |

---

## Структура проекта

```
user_data/
├── strategies/        — ваши торговые стратегии
├── config/            — конфигурация (не редактировать вручную)
├── data/              — исторические данные
├── logs/              — логи бота
└── backtest_results/  — результаты бэктестов
```
