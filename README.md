# Algo Trading Academy
> Инфраструктура кванмингового бота на базе Freqtrade

Добро пожаловать в репозиторий курса по алгоритмической торговле. Вся основная обучающая информация находится на платформе курса. Этот репозиторий содержит готовую и настроенную инфраструктуру для работы.

## 🚀 Как начать работу

Вам не нужно вручную редактировать сложные конфигурации для запуска. В вашем редакторе (Cursor) встроен персональный ИИ-ассистент, который уже обучен управлять этой инфраструктурой.

Просто откройте чат (Cmd/Ctrl + L) и напишите:
> **"Агент, запусти проект"**

Ассистент сам проверит Docker, настроит порты и поднимет интерфейс для работы.

## 📁 Доступные модули

Платформа поддерживает два независимых режима работы:
1. **Базовый (Криптовалюты):** Работает из коробки (Binance, Bybit и т.д.) по команде *«Запусти проект»*.
2. **Продвинутый (Фондовый рынок):** Торговля акциями и фьючерсами через Interactive Brokers. Для запуска напишите агенту: *«Запусти модуль Interactive Brokers»*. 
   *(Требуется наличие счета и прописанные данные `TWS_USERID` в файле `.env`)*.
