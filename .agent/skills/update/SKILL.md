# Обновление проекта

## Алгоритм

### 1. Проверь наличие локальных изменений

```
git status
```

Если есть незакоммиченные изменения в **стратегиях** (`user_data/strategies/`) — сохрани их:
```
git stash
```

### 2. Обнови

```
git pull origin main
```

### 3. Верни локальные изменения (если были)

```
git stash pop
```

Если возник конфликт — сообщи ученику и помоги решить.

### 4. Ответь кратко

> ✅ Проект обновлён до последней версии.

Если были обновлены скиллы или правила — перечитай `.project-context`.
